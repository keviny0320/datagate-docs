## API Key

```bash
curl -H "Authorization: Bearer dg_live_..." \
  https://api.datagate.dev/v1/datasets/discover
```

### Generate API Key

```bash
curl -X POST -H "Authorization: Bearer <jwt>" \
  https://api.datagate.dev/v1/auth/api-keys
```

Response (key shown once):
```json
{"api_key": "dg_live_abc123..."}
```

### Revoke API Key

```bash
curl -X DELETE -H "Authorization: Bearer <jwt>" \
  https://api.datagate.dev/v1/auth/api-keys
```

## JWT Login Flow

### 1. Login

```bash
curl -X POST https://api.datagate.dev/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "user@example.com", "password": "..."}'
```

Response:
```json
{"token": "<temporary_token>", "message": "verification code sent"}
```

### 2. Verify 2FA

```bash
curl -X POST https://api.datagate.dev/v1/auth/login/verify \
  -H "Content-Type: application/json" \
  -d '{"token": "<temporary_token>", "code": "123456"}'
```

Response:
```json
{"access_token": "eyJ..."}
```

Also sets an HttpOnly `refresh_token` cookie.

### 3. Refresh

```bash
curl -X POST https://api.datagate.dev/v1/auth/refresh \
  --cookie "refresh_token=..."
```

Access tokens expire after 15 minutes.
